⚠️ This is the static preview. The live dashboard requires the Python backend to be running.
How to start:
1
Add your API key to .env file:
ANTHROPIC_API_KEY=sk-ant-...
Get it at: console.anthropic.com
2
Double-click start.py or run in terminal:
python start.py
3
Browser opens automatically at:
http://localhost:8000/app
4
Type any psychology/neuroscience topic, click Start Research, and watch the AI work!
